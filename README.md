## Databases :

- To show available databases :
```bash
  show dbs
```
- To use (create) a database :
```bash
  use [db-name]  
``` 
- To add a collection :
```bash
  db.createCollection("[db-name]")  
```
- To drop a the in use database :
```bash
  db.dropDatabase()   
```
